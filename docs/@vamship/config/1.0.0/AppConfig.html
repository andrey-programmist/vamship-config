AppConfig

AppConfig

Wrapper for an application configuration object, based on configuration data passed as a constructor parameter. Exposes methods to allow queryig of data using dot separated property names.

This class is not meant to be instantiated directly, but will be returned from a call to config.getConfig.

Returns the value of the requested configuration property. Properties can be specified as dot separated strings to reference nested properties within complicated object structures.
